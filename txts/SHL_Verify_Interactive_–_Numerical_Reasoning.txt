Overview 

The Verify Interactive suite of assessments, including Numerical Reasoning, is a set of assessments 
designed using mobile-first technology. These assessments offer an interactive, business-relevant 
approach to predict workplace performance using engaging, activity-based assessment design that can 
be deployed on any device type, including mobile phones.  

Verify Interactive – Numerical Reasoning is an assessment used for job candidates applying to mid-
level positions that require numerical reasoning ability. Sample tasks for jobs that may require numerical 
reasoning include but are not limited to analysis and interpretation of numerical data (e.g., budgets & 
costs, trends and margins). Potential roles that use this ability are Finance, IT, Professional Services, 
Production and Sales.  

The Verify Interactive – Numerical Reasoning test utilizes computer adaptive technology, offering the 
following benefits: 

•  Shorter testing administration times 

•  Ability to offer the test in an unsupervised setting 

•  A more precise score
